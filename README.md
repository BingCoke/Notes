## 我的知识库

### 知识库，组的建立

可以自己建立知识库，知识库中的组，删除，修改，点击下面对应的按钮即可，

![image-20210411083232234](README.assets/image-20210411083232234.png)

### 笔记的建立

建立知识库和组之后可以新建一个笔记

点击增加按钮即可

![image-20210411084043081](README.assets/image-20210411084043081.png)

![image-20210411083552394](README.assets/image-20210411083552394.png)

#### 图片的增加

点击插入图片按钮，选择图片就可以将想要插入的图片插入

![image-20210411083653612](README.assets/image-20210411083653612.png)

![image-20210411083713111](README.assets/image-20210411083713111.png)

可以对图片选中之后剪贴，将光标指向任意位置再粘贴就可以把图片放入任何位置

![image-20210411083818038](README.assets/image-20210411083818038.png)

#### 页数

作者可以自由控制页数

![image-20210411083912186](README.assets/image-20210411083912186.png)

点击增加和减少页数的按钮即可

#### 公开

点击底部公开的小按钮就可以对笔记选择公开或者不公开

#### 保存

修改完，记得要保存

![image-20210411084249632](README.assets/image-20210411084249632.png)
